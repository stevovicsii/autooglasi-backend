Auto Oglasi - Backend (REST API)
Ovaj repozitorijum sadrži serverski deo klijent-server aplikacije za postavljanje i pregled auto oglasa. Aplikacija je razvijena u Spring Boot radnom okviru i implementira kompletnu RESTful arhitekturu.

Ključne funkcionalnosti:

Relaciona baza podataka (MySQL) sa 5 tabela.

Kompletne CRUD operacije nad entitetima.

Bezbednost sistema primenom Spring Security-ja i JWT (JSON Web Token) tehnologije za autentifikaciju i autorizaciju.

Sistem automatskog osvežavanja tokena (Refresh Token).
